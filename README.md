# 🤖 Autonomous Multi-Source Web Research Agent

A production-ready, asynchronous intelligence scraping and analysis workspace built using **Python**, **Agno AI Framework**, and **Streamlit**. This workspace implements autonomous agent loop architecture to orchestrate live web search tool compilation, raw article filtering, and structured markdown intelligence reporting dynamically from consumer inputs.

## 🚀 Live Application URL
👉 [Launch Live Agent Instance on Streamlit Cloud](https://share.streamlit.io/) *(Update with your live URL once deployed)*

## ✨ Architectural Core Features
* **Autonomous Search Instrumentation:** Integrates native `DuckDuckGoTools` matrix nodes, allowing the Large Language Model to control real-time crawling tasks over live internet parameters.
* **Insulated State Secret Vaulting:** Incorporates client-side side-panel dynamic token parsing inputs ensuring zero-retention runtime memory management for OpenAI secret keys.
* **Contextual Prompt Rigging:** Deploys structured executive parameter instructions instructing the agent framework to format outputs with explicit content hierarchies, markdown matrices, and source citations.
* **Deterministic Exception Routing:** Wrapped in localized `try-except` telemetry rings to isolate execution errors safely without crashing the system runtime layout.

## 🛠️ System Engineering Stack
* **Agent Core Engine Framework:** Agno (Phidata implementation fork)
* **Underlying Inference Model:** OpenAI `gpt-4o-mini`
* **Frontend Controller UI:** Streamlit Web Server Layout
* **Scraping Pipeline Node:** DuckDuckGo Search Engine Utility API
